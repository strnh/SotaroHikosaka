# チェロ弾き
* 1983年に楽器手をにした。
* 1986年頃から市民オーケストラなどに参加。
* 1997年ー2001年育児で中断
* 2002年、ミューザ川崎の市民オーケストラによる柿落としに参加。ベートーヴェン交響曲第９番、
* 2011年、古典専門のオーケストラに参加。
* 有名ではない作曲家の作品の演奏会が多い
